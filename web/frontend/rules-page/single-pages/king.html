Король может двигаться только на одно поле в любом направлении - вверх, вниз, в стороны, по диагонали. Нажмите на '>' кнопку на диаграмме ниже, чтобы увидеть, как король может передвигаться по доске. Король не может ходить на те поля, на которых он будет находиться под шахом (то есть может быть взят).
